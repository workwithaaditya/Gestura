# Optional Textures for Better Looks

You can drop texture images here to enhance the built-in shapes.

Supported files (optional):

- Earth (sphere):
  - earth_daymap.jpg (diffuse/color)
  - earth_normal.jpg (normal map)
  - earth_specular.jpg (specular/metalness map)

- Crate (cube):
  - crate_diffuse.jpg
  - crate_normal.jpg

- Ice cream cone (cone):
  - waffle.jpg

Place images in this folder and refresh the page. If a file is missing, the app will automatically use pretty fallback colors/materials.
